# Validador de ISBN – Proyecto de Pruebas Básicas

Este proyecto implementa un validador de ISBN-10 e ISBN-13 en Python y una suite completa de pruebas unitarias que aplican técnicas de caja negra, caja blanca, particiones de equivalencia, análisis de fronteras y property-based testing.


## Ejecución local
```bash
pytest --cov=src --cov-branch --cov-report=html
```

### Clonar el repositorio
```bash
git clone https://github.com/davidgarcia57/validador-isbn.git
cd validador-isbn
```
